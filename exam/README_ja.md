# ExamSystem 使用説明書

## システム要件
- Java 8 以上
- OpenGL 対応グラフィックス環境（Linux: mesa-utils をインストール）

## 起動方法
./start-exam.sh
または
java -jar ExamSystem.jar

## 機能
- 学習プランの選択（8つの系統的モジュール）
- 多様な問題形式：コーディング、選択、穴埋め、マッチング、デバッグ、補完
- 自動保存
- 実績システムとスター報酬
- 多言語UI（7言語切り替え可能）
- 内蔵コードエディタ（シンタックスハイライト、自動補完）

## 初回使用
1. ウェルカム画面表示
2. 「プランをインポート」をクリックして公式プランを読み込む（またはスキップ）
3. 難易度を選択（初級/中級/上級）
4. 問題を開始

## ショートカット
- Ctrl+S：進捗保存
- Ctrl+R：コードチェック実行
- Ctrl+←/→：前の問題/次の問題
- Esc：セクション選択に戻る

## 設定ファイル
進捗データは .exam_progress.json、.exam_unlocks.json、.exam_achievements.json に保存されます。

## トラブルシューティング
- ネイティブライブラリが見つからない場合、OpenGL ドライバがインストールされていることを確認してください。
- 起動できない場合、Java バージョンを確認：java -version
