# ExamSystem Руководство пользователя

## Системные требования
- Java 8 или выше
- Графическая среда с поддержкой OpenGL (для Linux установите mesa-utils)

## Запуск
./start-exam.sh
или
java -jar ExamSystem.jar

## Возможности
- Выбор плана обучения (8 систематических модулей)
- Типы заданий: программирование, выбор, заполнение, соответствие, отладка, дополнение
- Автосохранение прогресса
- Система достижений и звёзд
- Многоязычный интерфейс (7 языков)
- Встроенный редактор кода (подсветка синтаксиса, автодополнение)

## Первое использование
1. Появится приветственный экран
2. Нажмите «Импорт плана», чтобы загрузить официальные планы (или пропустите)
3. Выберите сложность (Начальный/Средний/Продвинутый)
4. Начните отвечать на вопросы

## Горячие клавиши
- Ctrl+S: Сохранить прогресс
- Ctrl+R: Проверить код
- Ctrl+←/→: Предыдущий/Следующий вопрос
- Esc: Вернуться к выбору раздела

## Файлы конфигурации
Прогресс хранится в .exam_progress.json, .exam_unlocks.json, .exam_achievements.json.

## Устранение неполадок
- Если отсутствует нативная библиотека, убедитесь, что установлены драйверы OpenGL.
- Если не запускается, проверьте версию Java: java -version
